## React Pokedex App

A mini-encyclopedia of Pokémon stats and types.

The following technologies were used for this sample application:

 - ReactJs (Hooks)
 - Redux (Persistent)
 - [Material UI](http://material-ui.com/)
 - Axios
 - [PokéAPI](https://pokeapi.co/)
 - LazyLoad


### Installation
```sh
npm install
npm start
```
